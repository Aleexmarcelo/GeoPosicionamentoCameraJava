## Funcionalidades ⚙️
✔️ Funcionalidade 1
GeoPosicionamento: Mostra em apenas um clique, a localização do aparelho em latitude e longitude.

✔️ Funcionalidade 2
Câmera: Apertando o botão câmera do aplicativo, o usuário armazena uma foto com data e hora.
